# Airbnb Paris Market Analysis

## Overview
This project analyzes the Airbnb market in Paris to identify potential investment opportunities. The analysis focuses on how location, property characteristics, and amenities influence price, demand, and revenue.

## Dataset
The dataset includes Airbnb listings in Paris with information such as:
- Neighbourhood
- Room type
- Price
- Number of reviews (demand proxy)
- Capacity
- Amenities

## Analysis Modules
The project includes several key analyses:

- **Location Analysis:** Identifies neighbourhoods with strong demand and balanced supply.
- **Capacity Analysis:** Examines how guest capacity affects price, demand, and revenue.
- **Room Type Analysis:** Compares performance of entire homes, private rooms, and shared rooms.
- **Amenities Analysis:** Evaluates how amenities influence pricing, demand, and overall revenue.

## Key Insights
- Listings with more amenities tend to achieve higher prices and stronger demand.
- Kitchen-related amenities significantly improve booking demand.
- Larger capacity properties (5+ guests) generate higher average revenue.
